## Half-Christmas-Tree Pattern Generator

Write a method/function that takes an integer and prints a "half-christmas-tree" pattern of the given height.

### Sample Output:

For `height = 3`:

```
+
-+
+-+
```

For `height = 5`:

```
+
-+
+-+
-+-+
+-+-+
```

For `height = 10`:

```
+
-+
+-+
-+-+
+-+-+
-+-+-+
+-+-+-+
-+-+-+-+
+-+-+-+-+
-+-+-+-+-+
```
